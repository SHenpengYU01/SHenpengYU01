1. 求1~n这n个数的最小公倍数。

- 思路：`lcm[i] = LCM(lcm[i-1], i)`，算是**动态规划**。
- 核心实现：

    ```cpp
    int lcm = 1;
    for(int i = 2; i <= n; i++){
        lcm = i*lcm/gcd(lcm, i);
    }
    ```

2. 给定一个点O和n个点的坐标，求n个点中距离O最近的m的个点。
